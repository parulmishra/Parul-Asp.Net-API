## Description

_This application uses the API exposed by GitHub, which returns my most starred repositories.

| Behavior  | Input  | Output  |
|---|---|---|
|1.  When the user clicks projects in the navbar a list of most starred repositories will display. | Input: > projects is clicked in navbar. | Output: List of most starred repos displays.
